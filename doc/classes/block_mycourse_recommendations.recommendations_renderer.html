render_recommendations()

render_recommendations(array  $recommendations) : string

Generates the HTML for the given recommendations. The maximum number of recommendations is defined by self::MAX_RECOMMENDATIONS constant.

Parameters

array $recommendations

Recommendations of the week and course.

Returns

string —

The formatted HTML for the recommendations; or string indicating that there are no recommendations, if $recommendations is empty.

get_resource_moduleid()

get_resource_moduleid(array  $instances, integer  $resourceid, integer  $courseid) : integer

Finds the module id from a resource instance, which is needed for the "get_cm($moduleid)" function, to get module's name.

For that, uses the courses instances information, retrieved using "get_fast_modinfo($courseid)" function.

Parameters

array $instances

Key is resource type name, and value is an array of each instance of the course of that type.
integer $resourceid

The resource we are finding the moduleid of.
integer $courseid

The course the module belongs to.

Returns

integer —

Module id.
