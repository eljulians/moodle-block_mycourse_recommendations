import_data()

import_data(object  $formdata, object  $coursefile, object  $usersfile, object  $logsfile, integer  $currentcourseid)

Receiving the three required CSVs, performs the importation of the data of a course: course info, enrolled users, and log info. The data is imported to a total of three different tables. All operations are done in a unique transaction: if something fails (a CSV does not respect the format, an importing course exists, etc.), nothing will be saved.

After the importation, associates the just created historic course with the current course, and sets this course as active and personalizable, just in case it was marked as inactive and/or not personalizable.

Parameters

object $formdata

The data submited in form.
object $coursefile

The CSV file with the information about the course.
object $usersfile

The CSV file with the information about the users enrolled in courses.
object $logsfile

The CSV file with the information about the log views of the users.
integer $currentcourseid

The course id of the course for which the csv is being imported.

Throws

\Exception

If something bad happened when trying to insert the data. The exception is thrown after doing the rollback.
