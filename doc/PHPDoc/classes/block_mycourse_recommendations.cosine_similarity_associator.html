cosine_similarity()

cosine_similarity(array  $vector1, array  $vector2) : double

Calculates the cosine similarity of two vectors, which will be the log views of a current user, and a historic user.

The formula is: cos_sim($v1, $v2) = $v1 · $v2 / ||$v1|| * ||$v2||.

Parameters

array $vector1

The log views of a user.
array $vector2

The log views of another user.

Returns

double —

The cosine similarity between the two vectors, a number between 0 and 1, being 1 the highest similarity.
